How to use
  1. Set the same row and column size as the LinkedIn puzzle.
  2. Choose Place Number and click each numbered dot location in order.
  3. Click Auto Solve to search for a full path that hits the numbers in order and fills every cell.
  4. Use the visual path or the directions list to try it on LinkedIn.
  5. Or skip steps 1–2: paste a screenshot in the AI Assist panel below and let GPT build the board for you.
